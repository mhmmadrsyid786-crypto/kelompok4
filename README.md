# kelompok4
Sistem Smart MBG adalah platform digital terintegrasi untuk mengelola program makan siang gratis di sekolah secara efisien, akurat, dan transparan. Sistem ini mengotomatisasi pemenuhan gizi siswa, stok bahan baku, dan distribusi logistik, sekaligus menjamin keamanan konsumsi lewat pencatatan riwayat alergi siswa secara real-time.
